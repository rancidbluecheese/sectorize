== Description ==

Sectorize is a WordPress plugin that transforms traditional author archives into **sector-based content hubs**. It includes:

- **Sector-Based Organization**: Group author content by sectors.
- **SEO Optimization**: Improve search engine visibility with built-in SEO features.
- **Structured Data**: Automatically add schema markup for rich snippets.
- **Custom Rewrite Rules**: Create custom URLs for sectors.
- **Easy Setup**: Works out-of-the-box with WordPress 6.0+.

---

== Installation ==

1. Upload the `sectorize` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Configure the plugin settings under **Settings > Sectorize**.

---

== Frequently Asked Questions ==

= How do I create a sector? =
Go to **Settings > Sectorize** and add a new sector.

= Does this plugin work with custom post types? =
Yes, Sectorize is compatible with custom post types.

= Can I customize the SEO settings? =
Yes, you can configure SEO options in the plugin settings.
